# Portfolio-Meta-Learning-Engine

📄 Quant Research Note
Proyecto desarrollado como investigación independiente en finanzas cuantitativas y sistemas adaptativos de inversión. SANTIAGO PEÑA DIAZ
Sistema cuantitativo diseñado para adaptar dinámicamente la asignación de portafolio mediante selección inteligente de estrategias basada en regímenes de mercado.

---

⚙️ Componentes del Sistema

- Detección de regímenes (GMM + PCA)
- Meta-learning para selección de estrategias
- Optimización de portafolio (MVO, Risk Parity, etc.)
- Evaluación dinámica basada en Sharpe Ratio

📊 Qué hace el modelo

- Identifica condiciones de mercado (volatilidad, correlación, tendencia)
- Evalúa múltiples estrategias simultáneamente
- Selecciona la mejor estrategia en cada momento
- Ajusta el portafolio de forma adaptativa

---

🎯 Enfoque Cuantitativo

Este sistema replica un framework utilizado en hedge funds:

Market Regime → Strategy Selection → Portfolio Allocation → Performance

---

🧠 Valor del Proyecto

- Integra machine learning con finanzas cuantitativas
- Simula toma de decisiones adaptativa
- Permite analizar performance bajo distintos regímenes

---

🚀 Aplicaciones

- Asset allocation dinámico
- Estrategias multi-factor
- Sistemas de trading adaptativo
